## DÉTAIL DES SECTIONS

### 1. Navbar
- À gauche : nom du sous-domaine en monospace (`ulrich.ivoire.io`)
- À droite : petit badge "Powered by ivoire.io 🇨🇮" (lien vers ivoire.io)
- Sticky au scroll
- Hauteur : 56px

### 2. Hero / Profil
- Photo de profil : ronde, 120px, bordure orange subtile
- Nom : H1, blanc, bold
- Titre + localisation : gris, regular
- Bio courte : 1-2 lignes, gris clair
- Icônes sociales : GitHub, LinkedIn, Twitter, Email (liens vers les profils)
- 2 boutons CTA :
  - "Me contacter" → scroll vers formulaire contact
  - "M'embaucher" → lien vers formulaire ou email

### 3. Compétences
- Tags/badges avec les technologies
- Style : petits rectangles arrondis, fond gris foncé, texte blanc
- Possibilité de colorer selon la techno (ex: Flutter = bleu, Dart = vert)
- Source : renseigné manuellement ou importé depuis GitHub

### 4. Projets
- Grille 2 colonnes (1 colonne sur mobile)
- Chaque card :
  - Image/screenshot (ou placeholder si pas d'image)
  - Nom du projet
  - Description (2 lignes max)
  - Tags technos
  - Stars GitHub (si importé)
  - Lien externe (repo ou démo)
- **Import depuis GitHub** : récupérer les repos pinnés via GitHub API
- Ordre : par date ou par popularité

### 5. Expérience (optionnel)
- Timeline verticale simple
- Chaque entrée : titre, entreprise, dates, description courte
- Renseigné manuellement par le dev dans son dashboard
- Section masquée si vide

### 6. Formulaire de Contact
- 3 champs : Nom, Email, Message
- Bouton : "Envoyer le message"
- Action : envoie un email au dev (via API Resend) ou stocke dans Supabase
- Protection anti-spam : honeypot + rate limiting

### 7. Footer
- "Portfolio propulsé par ivoire.io 🇨🇮"
- Lien : "Crée le tien gratuitement →" (redirige vers ivoire.io)
- **Important** : ce footer est le moteur viral — il converting les visiteurs en nouveaux inscrits

## DONNÉES NÉCESSAIRES (profil)

```typescript
interface DevProfile {
  slug: string;           // "ulrich" → ulrich.ivoire.io
  full_name: string;      // "Ulrich Kouamé"
  title: string;          // "Lead Developer"
  city: string;           // "Abidjan"
  bio: string;            // Bio courte (max 200 chars)
  avatar_url: string;     // URL photo de profil
  github_url?: string;
  linkedin_url?: string;
  twitter_url?: string;
  email: string;
  skills: string[];       // ["Flutter", "Dart", "Firebase"]
  projects: Project[];
  experiences?: Experience[];
  is_available: boolean;  // Dispo pour missions
}

interface Project {
  name: string;
  description: string;
  image_url?: string;
  tech_stack: string[];
  github_url?: string;
  demo_url?: string;
  stars?: number;
}

interface Experience {
  role: string;
  company: string;
  start_date: string;
  end_date?: string;      // null = "Présent"
  description: string;
}
```
