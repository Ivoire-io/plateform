# 🎨 CATALOGUE DE TEMPLATES — ivoire.io

> **10 templates** disponibles pour les pages publiques des utilisateurs  
> Chaque type (dev, startup, entreprise, autre) peut choisir parmi tous les templates  
> **Sélection via** : Dashboard → Onglet Template → Prévisualisation en temps réel  
> **Technique** : Rendu conditionnel côté serveur — même données, mise en page différente

## RÉSUMÉ DES 10 TEMPLATES

| # | Nom | Style | Idéal pour | Mood |
|---|-----|-------|-----------|------|
| 1 | **Classic** | Page complète scrollable | Tous | Professionnel, polyvalent |
| 2 | **Minimal** | Très épuré, centré | Dev minimalistes | Zen, clean |
| 3 | **Bento** | Grille modulaire | Dev créatifs | Moderne, Apple-like |
| 4 | **Terminal** | Esthétique CLI/code | Dev backend/DevOps | Geek, technique |
| 5 | **Magazine** | Mise en page éditoriale | Freelancers, portfolios riches | Storytelling |
| 6 | **Timeline** | Chronologie verticale | Profils expérimentés | Carrière, parcours |
| 7 | **Card Stack** | Cartes empilées | Dev full-stack | Dynamique, interactif |
| 8 | **Split** | Écran divisé 50/50 | Profils visuels | Bold, design |
| 9 | **Startup Landing** | Landing page produit | Startups, fondateurs | Pitch, conversion |
| 10 | **Corporate** | Vitrine entreprise | Entreprises, agences | Sérieux, confiance |

## PERSONNALISATION

Chaque template supporte les options suivantes (Dashboard → Paramètres → Apparence) :

| Option | Détail | Valeurs |
|---|---|---|
| **Couleur d'accent** | Couleur des boutons, liens, badges | Prédéfinis (8 couleurs) ou personnalisé (hex) |
| **Mode** | Thème clair ou sombre | `light` / `dark` / `auto` (suit le visiteur) |
| **Image de fond** | Hero / header background | Upload ou aucune |
| **Ordre des sections** | Drag & drop des sections | Projets, Expériences, Contact, etc. |
| **Sections visibles** | Masquer des sections | Toggle par section |
| **Police** | Choix de typographie | Inter / Geist / JetBrains Mono / System |

## RÈGLES TECHNIQUES

### Architecture de rendu

```typescript
// portfolio/[slug]/page.tsx — Server Component
async function PortfolioPage({ params }) {
  const profile = await getProfile(params.slug);
  const template = profile.template ?? "classic";

  // Rendu conditionnel du template
  switch (template) {
    case "classic":   return <TemplateClassic profile={profile} />;
    case "minimal":   return <TemplateMinimal profile={profile} />;
    case "bento":     return <TemplateBento profile={profile} />;
    case "terminal":  return <TemplateTerminal profile={profile} />;
    case "magazine":  return <TemplateMagazine profile={profile} />;
    case "timeline":  return <TemplateTimeline profile={profile} />;
    case "cardstack": return <TemplateCardStack profile={profile} />;
    case "split":     return <TemplateSplit profile={profile} />;
    case "startup":   return <TemplateStartup profile={profile} />;
    case "corporate": return <TemplateCorporate profile={profile} />;
    default:          return <TemplateClassic profile={profile} />;
  }
}
```

|---|---|---|
