# Formules de souscription dev — Proposition revisee

> Strategie de monetisation adaptee au marche ivoirien pour les developpeurs
> Date : Mars 2026

---

## 1. Contexte et principes

### Realites du marche cible

- **Salaire moyen dev en CI** : 200 000 - 600 000 FCFA/mois (junior a senior)
- **Seuil psychologique** : Au-dela de 5 000 FCFA/mois, un dev junior hesite
- **Mobile money** : Methode de paiement dominante (Wave, Orange Money)
- **Freelance = minorite** : La majorite des devs sont salaries, le freelance est emergent
- **Valeur percue** : Un portfolio en ligne a peu de valeur percue seul. Il faut ajouter du concret.

### Principes directeurs

1. **Free doit etre genereux** : Le dev gratuit est l'ambassadeur viral de la plateforme
2. **Le payant doit resoudre un vrai probleme** : visibilite, recrutement, credibilite
3. **Eviter le "paywall de la honte"** : Ne jamais bloquer le profil publique derriere un paywall
4. **Monetiser la valeur, pas l'hebergement** : Le portfolio gratuit est un produit d'appel
5. **Penser annuel** : Les paiements mensuels en FCFA generent trop de friction (reabonnement manuel)

## 3. Comparatif avec les plans actuels

### Ce qui change

| Actuel (DB) | Propose | Changements |
|-------------|---------|-------------|
| Free (0) | Free (0) | Pas de changement majeur |
| Builder (3 000/mois) | Starter (2 000/mois ou 20 000/an) | Prix baisse + option annuelle |
| Pro (60 000/an) | Pro (15 000/an) | **Prix divise par 4** — l'ancien prix est trop eleve |
| Growth (custom) | Expert (contact) | Renomme, cible plus claire |
| Startup (10 000/mois) | Garder pour type=startup | Pas pour les devs |

### Justification de la baisse du Pro

- 60 000 FCFA/an = 91 EUR. C'est le prix d'un SaaS occidental, pas adapte au marche CI.
- A 15 000/an, on vise un volume 5-10x superieur, compensant largement la baisse de prix unitaire.
- Le cout marginal d'un dev supplementaire sur la plateforme est quasi-nul (pas d'infra dediee).

## 6. Strategie de conversion

### Funnel Free -> Starter

1. **Declencheur** : Le dev voit qu'il a eu des visites mais n'a pas les details → "Debloque tes stats detaillees"
2. **Friction** : Les 3 templates gratuits sont bons mais limites → "Decouvre 7 templates en plus"
3. **Urgence** : "Tu as recu 3 demandes de contact ce mois — affiche ton tarif pour filtrer les demandes"
4. **Social proof** : "128 devs ont deja un plan Starter"

### Funnel Starter -> Pro

1. **Declencheur** : Le dev freelance veut le badge verifie pour credibiliser ses propositions
2. **Friction** : Limite de 15 projets atteinte
3. **Valeur** : Export CV PDF + visibilite prioritaire = outil concret de prospection
4. **ROI** : "1 mission freelance = 10x le prix du Pro annuel"

### Mecanismes d'engagement

| Mecanisme | Implementation |
|-----------|---------------|
| Email hebdo "Tes stats" | Envoyer chaque lundi un resume : X vues, Y messages, position dans l'annuaire |
| Notification "Nouveau message" | Push email a chaque message recu sur le portfolio |
| Reminder profil incomplet | Envoyer 1 email apres 3 jours si le profil est < 80% complet |
| Parrainage double | Doubler les rewards pendant les 2 premiers mois de lancement |
| Essai gratuit 7j | Le Starter est essayable gratuitement pendant 7 jours |

## 8. Comparaison avec la concurrence

### Positionnement

| Plateforme | Prix portfolio | Marche | Avantage ivoire.io |
|------------|---------------|--------|-------------------|
| GitHub Pages | Gratuit | Global | Pas de portfolio "metier", pas d'annuaire, pas de sous-domaine |
| Vercel / Netlify | Gratuit | Global | Demande des competences techniques poussees |
| Linktree Pro | ~8 EUR/mois | Global | Pas de portfolio complet, pas d'annuaire |
| Bento.me | Gratuit | Global | Pas localise, pas d'ecosysteme startup |
| Read.cv | Freemium | Global | Pas localise, pas d'annuaire, pas d'ecosysteme |
| **ivoire.io** | **Gratuit -> 15 000/an** | **CI / AO** | **Sous-domaine, annuaire, ecosysteme startup, localise, mobile money** |

### Avantage competitif unique

1. **Localisation** : En francais, adapte au marche ivoirien et ouest-africain
2. **Ecosysteme** : Pas juste un portfolio — liaison devs/startups/entreprises
3. **Sous-domaine** : `ulrich.ivoire.io` est plus memorable que `github.io/ulrich`
4. **Mobile money** : Pay as they pay — Wave, Orange Money, Moov
5. **Communaute** : L'annuaire cree un reseau, pas juste un outil individuel
