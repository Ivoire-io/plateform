## 1. LOGO PRINCIPAL — ivoire.io

### Prompt Midjourney / DALL-E 3

```
Minimalist modern logo for a tech platform called "ivoire.io".
The logo should be a clean wordmark "ivoire.io" in lowercase.
Style: Silicon Valley tech startup aesthetic, similar to Vercel or Stripe logos.
Typography: geometric sans-serif font, clean and bold.
The ".io" part should use a subtle accent color (orange #FF6B00 or green #00A651).
Background: transparent or dark (#0A0A0A).
No icons, no illustrations — pure typography.
Colors: white text on dark background, with orange or green accent on ".io".
Professional, premium, authoritative. Represents Ivory Coast + technology.
--ar 3:1 --style raw --v 6
```
